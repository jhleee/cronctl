## How the examples were captured

The command/output blocks in this wiki were generated by actually running the current CLI.

- `CRONCTL_CRONTAB_BIN` was pointed at a fake local crontab script so the examples did not modify the real user crontab.
- Notification examples were sent to a local test webhook on `127.0.0.1`.
- The demo home used `/tmp/cronctl-docs-home`.

That setup makes the examples realistic while still being safe to reproduce locally.
